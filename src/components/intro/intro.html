Library of web UI components with minimal styles and dependencies, useful as starter code to build upon and customize.

Select a component from the left navigation menu. To see the code for each component, click the See Code button on the header. On the modal box, you can switch between HTML, CSS, SCSS, SASS, and JS.

Targeted browser support: Edge, Firefox, Chrome, Safari

To run this demo site locally:

  • Clone repository: git clone https://github.com/majoflosa/ui-components.git
  • Change to project directory: cd ui-components
  • Install dependencies: npm install
  • Start server: npm start
  • Open your browser to http://localhost:8080
